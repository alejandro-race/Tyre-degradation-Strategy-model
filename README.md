# Tyre Degradation & Pit Window Optimization Model

This project develops a simplified MATLAB-based race strategy model to evaluate tyre degradation, fuel burn-off and one-stop pit strategy optimization.

## Objective

The objective is to compare valid one-stop tyre strategies and identify the optimal pit stop lap by minimizing total simulated race time.

## Model Features

- Non-linear tyre degradation model
- Fuel burn-off effect
- Fuel-load sensitivity
- Tyre cliff penalty
- One-stop strategy comparison
- Pit window optimization
- Strategy recommendation based on total race time

## Tools

- MATLAB
- Data visualization
- Motorsport performance analysis

## Outputs

The model generates:
- Tyre degradation curves
- Pit window optimization plot
- Best strategy lap time profile
- Strategy comparison chart

## Disclaimer

This is a simplified engineering model. The parameters are assumed and are not based on confidential team or official tyre supplier data.
